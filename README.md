Общее описание
Эта программа — лабораторный проект по компьютерному зрению на языке Python. Она умеет в реальном времени (через веб-камеру) и на статических фотографиях находить лица, глаза и улыбки.
Главная особенность программы — возможность накладывать AR-маску (дополненную реальность) на лицо пользователя или рисовать рамки вокруг обнаруженных объектов. Программа также умеет считать количество людей в кадре,
замерять производительность (FPS) и тестировать точность работы алгоритмов.

Архитектура и используемые модели (Техническая часть)
Важно понимать, что эта программа не использует глубокие нейронные сети (как, например, YOLO или TensorFlow), а базируется на классических алгоритмах компьютерного зрения.
   Модель детекции: Каскады Хаара (Haar Cascades)
Программа использует технологию Viola-Jones, которая была стандартом индустрии до прихода нейросетей.
Как это работает: Алгоритм использует "скользящее окно", которое проходит по изображению с разным масштабом. Внутри окна анализируются простые признаки (например, область глаз обычно темнее области щек).
Файлы моделей (.xml):
haarcascade_frontalface_default.xml — обученная модель для поиска лиц (анфас).
haarcascade_eye.xml — модель для поиска глаз (работает только внутри области лица).
haarcascade_smile.xml — модель для поиска улыбок.
Плюсы: Очень быстрая работа, не требует мощной видеокарты, работает на обычных процессорах.
Минусы: Может не сработать, если лицо сильно повернуто в профиль или закрыто.
   Алгоритм наложения маски (Alpha Blending)
Если маска найдена, программа использует математику прозрачности (альфа-канал):
Маска растягивается до размера найденного лица.
Каждый пиксель лица на видео смешивается с пикселем маски по формуле:
Итоговый_цвет = (Цвет_Видео * (1 - Прозрачность)) + (Цвет_Маски * Прозрачность)
Это позволяет маске "приклеиться" к лицу и выглядеть естественно.


Системные требования и Установка
Что нужно для запуска:
Установленный Python 3.7 или выше.
Веб-камера (для режима 2 и 3).
Свободное место на диске для сохранения фото.

Шаг 1: Установка библиотек
Откройте командную строку (Terminal) и введите:
pip install opencv-python numpy

Шаг 2: Подготовка файлов
Создайте папку для проекта и положите в нее:
Сам файл скрипта (назовите его main.py).
Файл маски: Найдите в интернете картинку mask.png (например, усы, очки или корона). Важно: у картинки должен быть прозрачный фон (альфа-канал).
(Опционально) XML-файлы каскадов. Если их нет в папке, программа попытается найти их в системных папках Python автоматически.

Как запустить программу
Откройте терминал в папке с проектом.
Введите команду:
python main.py
В консоли появится меню. Выберите цифру и нажмите Enter.

Описание режимов работы (Функционал)
Режим 1: Обработка статичных фото (Пункт 2)
Что делает: Программа сканирует папку, берет все файлы .jpg и .png, находит на них лица и глаза, накладывает маску.
Результат: В папке проекта появится новая папка results_static. Там будут сохранены обработанные фотографии.
Визуализация:
На лице — ваша маска (или зеленая рамка, если маски нет).
На глазах — синие прямоугольники.

Режим 2: Веб-камера в реальном времени (Пункты 3, 5, 6)
Что делает: Открывает поток с камеры.
Функции:
Зеркальное отображение: Картинка переворачивается, как в зеркале.
Детекция: Ищет лица, внутри лиц ищет глаза и улыбки.
Счетчик людей: В левом нижнем углу пишет People: X. Если человек уходит или заходит, всплывает желтая надпись +1 или -1.
FPS: В правом верхнем углу показывает скорость обработки кадров (Frames Per Second).
AR: Накладывает маску на лицо.
Умная фильтрация: Если алгоритм нашел 3 глаза (ошибка), программа оставит только 2 самых крупных, чтобы не рисовать "третий глаз" на лбу.
Управление: Нажмите клавишу q или Esc, чтобы закрыть программу.

Режим 3: Тест параметров (Бенчмарк)
Что делает: Это инструмент для исследования. Программа запускает камеру и 3 секунды тестирует разные настройки детекции (scaleFactor и minNeighbors).
Результат: В консоль выводится таблица.
Scale: Насколько уменьшается картинка при поиске (влияет на скорость и размер лиц).
MinN: Сколько "соседей" нужно для подтверждения, что это лицо (влияет на ложные срабатывания).
Avg FPS: Средняя скорость.
Avg Faces: Сколько лиц нашел в среднем.
Зачем это нужно: Чтобы понять, какие настройки лучше подходят для вашей конкретной камеры и компьютера.

Интерпретация результатов
Зеленая рамка / Картинка на лице
Лицо обнаружено. Если маска не наложилась, а рамка есть — проверьте файл mask.png.

Синие квадраты на глазах
Глаза найдены внутри лица.

Красные квадраты на рте
Обнаружена улыбка.

People: 0
Лиц не найдено. Попробуйте подойти ближе или улучшить свет.

FPS < 15
Программа работает медленно. Возможно, камера дает слишком высокое разрешение или компьютер перегружен.

Возможные ошибки и их решение
ModuleNotFoundError: No module named 'cv2'
Не установлена библиотека
Выполните pip install opencv-python

Не удалось открыть камеру
Камера занята другой программой (Zoom, Skype) или нет прав доступа
Закройте другие программы. На macOS зайдите в Настройки -> Конфиденциальность -> Камера и разрешите доступ Терминалу/Python.

Маска не появляется, только рамка
Файл mask.png лежит в другой папке или сохранен без прозрачности (JPG вместо PNG)
Положите PNG с прозрачным фоном рядом со скриптом.

FileNotFoundError: Не удалось загрузить каскады
Программа не нашла XML-файлы
Скачайте стандартные каскады OpenCV с GitHub и положите их в папку со скриптом.

Лица не находятся
Плохое освещение или лицо в профиль
Повернитесь прямо к камере, включите свет.

Советы по улучшению работы
Свет: Каскады Хаара очень чувствительны к свету. Лучше всего, чтобы источник света был спереди, а не сзади (против света программа работает плохо).
Настройка параметров: Если программа "тормозит" (низкий FPS), откройте файл main.py и измените в начале настройки:
DEFAULT_SCALE = 1.3 (быстрее, но ищет только крупные лица).
DEFAULT_MIN_NEIGHBORS = 8 (меньше ложных срабатываний, но строже).
Маска: Для лучшего эффекта AR используйте маску, которая по пропорциям похожа на лицо (примерно 100x100 пикселей).
