ОБЩЕЕ ОПИСАНИЕ

Эта программа — лабораторный проект по компьютерному зрению на языке Python. 
Она умеет в реальном времени (через веб-камеру) и на статических фотографиях 
находить лица, глаза и улыбки.

Главная особенность программы — возможность накладывать AR-маску 
(дополненную реальность) на лицо пользователя или рисовать рамки вокруг 
обнаруженных объектов. Программа также умеет считать количество людей в кадре,
замерять производительность (FPS) и тестировать точность работы алгоритмов.



 АРХИТЕКТУРА И ИСПОЛЬЗУЕМЫЕ МОДЕЛИ (ТЕХНИЧЕСКАЯ ЧАСТЬ)


 ВАЖНО ПОНИМАТЬ: эта программа не использует глубокие нейронные сети 
(как, например, YOLO или TensorFlow), а базируется на классических 
алгоритмах компьютерного зрения.

 МОДЕЛЬ ДЕТЕКЦИИ: КАСКАДЫ ХААРА (Haar Cascades)
--------------------------------------------------

Программа использует технологию Viola-Jones, которая была стандартом 
индустрии до прихода нейросетей.

КАК ЭТО РАБОТАЕТ:
Алгоритм использует "скользящее окно", которое проходит по изображению 
с разным масштабом. Внутри окна анализируются простые признаки 
(например, область глаз обычно темнее области щек).

ФАЙЛЫ МОДЕЛЕЙ (.xml):

• haarcascade_frontalface_default.xml — обученная модель для поиска лиц 
  (анфас).
  
• haarcascade_eye.xml — модель для поиска глаз (работает только внутри 
  области лица).
  
• haarcascade_smile.xml — модель для поиска улыбок.

ПРЕИМУЩЕСТВА:
 Очень быстрая работа
 Не требует мощной видеокарты
 Работает на обычных процессорах

НЕДОСТАТКИ:
 Может не сработать, если лицо сильно повернуто в профиль или закрыто


АЛГОРИТМ НАЛОЖЕНИЯ МАСКИ (Alpha Blending)
--------------------------------------------

Если маска найдена, программа использует математику прозрачности 
(альфа-канал):

1. Маска растягивается до размера найденного лица.
2. Каждый пиксель лица на видео смешивается с пикселем маски по формуле:

   Итоговый_цвет = (Цвет_Видео * (1 - Прозрачность)) + 
                   (Цвет_Маски * Прозрачность)

Это позволяет маске "приклеиться" к лицу и выглядеть естественно.



 СИСТЕМНЫЕ ТРЕБОВАНИЯ И УСТАНОВКА


ЧТО НУЖНО ДЛЯ ЗАПУСКА:
 Установленный Python 3.7 или выше.
 Веб-камера (для режима 2 и 3).
 Свободное место на диске для сохранения фото.


ШАГ 1: УСТАНОВКА БИБЛИОТЕК
------------------------------

Откройте командную строку (Terminal) и введите:

    pip install opencv-python numpy


ШАГ 2: ПОДГОТОВКА ФАЙЛОВ
----------------------------

Создайте папку для проекта и положите в нее:

1. Сам файл скрипта (назовите его main.py).
2. Файл маски: Найдите в интернете картинку mask.png (например, усы, очки 
   или корона). 
    ВАЖНО: у картинки должен быть прозрачный фон (альфа-канал).
3. (Опционально) XML-файлы каскадов. Если их нет в папке, программа попытается 
   найти их в системных папках Python автоматически.



КАК ЗАПУСТИТЬ ПРОГРАММУ


1. Откройте терминал в папке с проектом.
2. Введите команду:

       python main.py

3. В консоли появится меню. Выберите цифру и нажмите Enter.



 ОПИСАНИЕ РЕЖИМОВ РАБОТЫ (ФУНКЦИОНАЛ)


РЕЖИМ 1: ОБРАБОТКА СТАТИЧНЫХ ФОТО (Пункт 2)
----------------------------------------------

ЧТО ДЕЛАЕТ:
Программа сканирует папку, берет все файлы .jpg и .png, находит на них 
лица и глаза, накладывает маску.

РЕЗУЛЬТАТ:
В папке проекта появится новая папка results_static. Там будут сохранены 
обработанные фотографии.

ВИЗУАЛИЗАЦИЯ:
• На лице — ваша маска (или зеленая рамка, если маски нет).
• На глазах — синие прямоугольники.


РЕЖИМ 2: ВЕБ-КАМЕРА В РЕАЛЬНОМ ВРЕМЕНИ (Пункты 3, 5, 6)
----------------------------------------------------------

ЧТО ДЕЛАЕТ:
Открывает поток с камеры.

ФУНКЦИИ:
 Зеркальное отображение: Картинка переворачивается, как в зеркале.
 
 Детекция: Ищет лица, внутри лиц ищет глаза и улыбки.
 
 Счетчик людей: В левом нижнем углу пишет People: X. Если человек уходит 
  или заходит, всплывает желтая надпись +1 или -1.
  
 FPS: В правом верхнем углу показывает скорость обработки кадров 
  (Frames Per Second).
  
 AR: Накладывает маску на лицо.
 
 Умная фильтрация: Если алгоритм нашел 3 глаза (ошибка), программа оставит 
  только 2 самых крупных, чтобы не рисовать "третий глаз" на лбу.

УПРАВЛЕНИЕ:
Нажмите клавишу q или Esc, чтобы закрыть программу.


РЕЖИМ 3: ТЕСТ ПАРАМЕТРОВ (БЕНЧМАРК)
---------------------------------------

ЧТО ДЕЛАЕТ:
Это инструмент для исследования. Программа запускает камеру и 3 секунды 
тестирует разные настройки детекции (scaleFactor и minNeighbors).

РЕЗУЛЬТАТ:
В консоль выводится таблица.

• Scale: Насколько уменьшается картинка при поиске (влияет на скорость 
  и размер лиц).
  
• MinN: Сколько "соседей" нужно для подтверждения, что это лицо (влияет на 
  ложные срабатывания).
  
• Avg FPS: Средняя скорость.

• Avg Faces: Сколько лиц нашел в среднем.

ЗАЧЕМ ЭТО НУЖНО:
Чтобы понять, какие настройки лучше подходят для вашей конкретной камеры 
и компьютера.



ИНТЕРПРЕТАЦИЯ РЕЗУЛЬТАТОВ
---------------------------------------

 ЗЕЛЕНАЯ РАМКА / КАРТИНКА НА ЛИЦЕ
   Лицо обнаружено. Если маска не наложилась, а рамка есть — проверьте 
   файл mask.png.

 СИННИЕ КВАДРАТЫ НА ГЛАЗАХ
   Глаза найдены внутри лица.

 КРАСНЫЕ КВАДРАТЫ НА РТЕ
   Обнаружена улыбка.

 PEOPLE: 0
   Лиц не найдено. Попробуйте подойти ближе или улучшить свет.

 FPS < 15
   Программа работает медленно. Возможно, камера дает слишком высокое 
   разрешение или компьютер перегружен.



 ВОЗМОЖНЫЕ ОШИБКИ И ИХ РЕШЕНИЕ


 ModuleNotFoundError: No module named 'cv2'
   --------------------------------------------
   ПРИЧИНА: Не установлена библиотека
   
   РЕШЕНИЕ: Выполните
            pip install opencv-python


 Не удалось открыть камеру
   --------------------------
   ПРИЧИНА: Камера занята другой программой (Zoom, Skype) или нет прав 
            доступа
   
   РЕШЕНИЕ: Закройте другие программы. На macOS зайдите в 
            Настройки -> Конфиденциальность -> Камера и разрешите доступ 
            Терминалу/Python.


 Маска не появляется, только рамка
   -----------------------------------
   ПРИЧИНА: Файл mask.png лежит в другой папке или сохранен без 
            прозрачности (JPG вместо PNG)
   
   РЕШЕНИЕ: Положите PNG с прозрачным фоном рядом со скриптом.


 FileNotFoundError: Не удалось загрузить каскады
   ------------------------------------------------
   ПРИЧИНА: Программа не нашла XML-файлы
   
   РЕШЕНИЕ: Скачайте стандартные каскады OpenCV с GitHub и положите их 
            в папку со скриптом.


 Лица не находятся
   ------------------
   ПРИЧИНА: Плохое освещение или лицо в профиль
   
   РЕШЕНИЕ: Повернитесь прямо к камере, включите свет.



 СОВЕТЫ ПО УЛУЧШЕНИЮ РАБОТЫ


 СВЕТ
   Каскады Хаара очень чувствительны к свету. Лучше всего, чтобы источник 
   света был спереди, а не сзади (против света программа работает плохо).


 НАСТРОЙКА ПАРАМЕТРОВ
   Если программа "тормозит" (низкий FPS), откройте файл main.py и измените 
   в начале настройки:
   
   • DEFAULT_SCALE = 1.3 
     (быстрее, но ищет только крупные лица).
   
   • DEFAULT_MIN_NEIGHBORS = 8 
     (меньше ложных срабатываний, но строже).


 МАСКА
   Для лучшего эффекта AR используйте маску, которая по пропорциям похожа 
   на лицо (примерно 100x100 пикселей).
